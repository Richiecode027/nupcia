Conecta su espacio

Sincronización en tiempo real para los dos. Configúralo una vez (gratis) y comparte el enlace con tu pareja.

  1. Entra a supabase.com/dashboard y crea un proyecto nuevo (ya tienes cuenta).
  2. Abre el SQL Editor, pega el script de abajo y presiona Run. Crea la tabla y activa el tiempo real.
  3. Ve a Project Settings → API y copia la Project URL y la clave anon public.
  4. Pégalas aquí abajo y presiona Conectar.
📋 Script SQL — cópialo y ejecútalo en Supabase 
create table if not exists espacios (
  id text primary key,
  data jsonb,
  updated_at timestamptz default now()
);

alter table espacios enable row level security;

drop policy if exists "acceso por enlace" on espacios;
create policy "acceso por enlace" on espacios
  for all using (true) with check (true);

do $$
begin
  alter publication supabase_realtime add table espacios;
exception when others then null;
end $$;
